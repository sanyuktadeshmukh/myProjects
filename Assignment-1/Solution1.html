Explanation


As we all know there is a format to each and ever piece of information which we use in our day today life, Eg: Book, Blog, Journal etc.

Need of structure: For each type of data there are unique ways to implant it in the code. A proper structure guides us and tells us where to place a particular part of code.

Example:
1. Information like title, main background, metadata, etc should be placed in between head tag.
2. Data which will be going to be displayed on website should be placed under body tag.

Basic Structure of Html file :

1. <html> : HTML tag works as a container containging our whole web page's code. It is the most outermost shell of the html file. HTML tag contains two major sub tags within itself, .i.e HEAD and BODY.
2. <head> : Head tag contains information regarding whole webpage. This includes title, background, metadata etc.
3. <body> : Body tag contains all content that the user will be going to consume. As per the type of content it contains different tags.
Example:
1.For displaying a headind we use <h1>, <h2>,...,<h6>.
2.For paragraph we use <p>.
3.For links we use <a>.
